# URL-Slicer

URL-Slice is used to extract main website link from a url.
Eg:-
    URL -> https://www.youtube.com/results?search_query=physics+practical+class+12
    Sliced URL -> www.youtube.com
    
Technologies used :
    1) Python 3.8.6
    2) Tkinter (for GUI)
